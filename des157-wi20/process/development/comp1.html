Composition #1

Expanding on the rough image created via the paper prototyping phase, I emphasized a linear design layout. The rules/lines in the design would aid in navigating the user towards key interactive parts. However, as I was designing this prototype, I realized that there will be layout struggles. For the next iterations, I aim for a more consistent navigation system and layout organization.
